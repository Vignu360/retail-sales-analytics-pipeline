# Retail Sales Analytics Pipeline

## 📌 Overview
End-to-end Azure–Databricks pipeline ingesting retail sales data, transforming with PySpark, storing in Synapse, and visualizing KPIs in Power BI. Demonstrates cloud ETL, analytics, and dashboarding with reproducible code and business impact.

## 🛠️ Tech Stack
- **Azure Blob Storage / ADLS Gen2** – raw data storage
- **Databricks Community Edition (PySpark)** – data transformation
- **Azure Synapse Analytics** – processed data warehouse
- **Power BI Desktop** – visualization and reporting
- **GitHub** – version control and documentation
